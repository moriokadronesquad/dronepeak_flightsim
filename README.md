# DRONEPEAK FLIGHT SIM ｜一等・二等 無人航空機 実地講習シミュレーター（試作 v0.1）

一等・二等 無人航空機操縦士の**実地講習（実技）練習用**ブラウザシミュレーターです（種目セレクタで一等3科目／二等3科目を切替）。
国土交通省航空局の実地試験 実施基準・実施細則（回転翼航空機マルチローター・**一等**／令和7年12月5日改正・令和8年6月5日施行）に基づき、コース・区画・採点を再現しています。修了審査は同細則「第VII章 準用」により実地試験に準じて実施します（受験者→受講者、試験員→修了審査員 等）。

> 本シミュレーターは練習補助を目的とした**試作版**であり、**実際の試験・修了審査の合否を保証するものではありません。**

---

## 動作環境

- 最新の **Google Chrome / Microsoft Edge**（WebGL・Gamepad API 対応）
- インストール不要・**完全オフライン動作**

## 使い方

1. `index.html` `dronepeak_ikkyu_flightsimV0.1.html` `three.min.js` `emblem.png` を**同じフォルダ**に置きます。
2. `index.html`（タイトル画面）を開き、配布された**アクティベーションキー**を入力して認証します。
3. 「シミュレーターを起動」を押すとシミュレーター本体が開きます。
4. ゲームパッド／プロポを接続すると自動認識します（未接続でもキーボードで操作可）。

※ ファイルは必ずセットで配置してください（HTMLが隣の `three.min.js` を読み込みます）。
> キー作成ツール（`_admin_keygen.html`）は管理用フォルダに別置きで、**GitHubには公開しません**。

## 操作

- **入力モード**：モード1／モード2（メニューで切替。**選んだモードは保存され、試験開始でモード2に戻りません**）
- **スティック割り当て**：標準プリセット、または「オート割り当て」で実機の軸を読み取って自動設定
- **キーボード**：スロットル `W/S`、エレベーター `↑/↓`、ラダー `A/D`、エルロン `←/→`、**Enter＝実地審査開始 / Space＝停止**
- **プロポ軸スイッチ**：軸4＝実地審査開始 / 軸5＝リセット

## 一等の科目（基本・25kg未満・全科目ATTI）

| 科目 | モード | 高度 | 制限時間 | 主な内容 |
|---|---|---|---|---|
| 高度変化を伴うスクエア飛行 | ATTI | 1.5m⇄3.5m | 6分 | B-C間・E-D間で高度変化、機首=進行方向 |
| ピルエットホバリング | ATTI | 3.5m | 3分 | 約20秒で1回転（許容16〜26秒） |
| 緊急着陸を伴う8の字飛行 | ATTI | 1.5m | 5分 | 円径5m、1周以上＋緊急事態のランダム宣言→緊急着陸 |

- **合格：各科目 80点以上**（二等は70点）。種目選択は二等3科目／一等3科目を切替可能。
- 種目で合格すると、約2.5秒後にプルダウンの**次の種目（上→下）へ自動切替**します。

## 主な機能

- 講師ナビ（音声＋テキスト案内）、推奨飛行ガイドライン、環境風（風速・風向き）
- 自動採点プレビュー（100点持ち点・減点式）、減点ホイッスル／不合格ブザー、制限時間カウントダウン
- 8の字の区画は公式図準拠（外周＝矩形、円の内側＝中心ディスクの減点）
- ピルエットの回転速度評価（16秒未満／26秒以上で減点）
- 緊急着陸：発生地点にホバリング用リング表示、帰還時は円の内側減点を無効化（外周減点・不合格区画は有効）
- 実地審査開始中は上部案内・右上メニューを半透過にして視界を確保
