# DRONEPEAK FLIGHT SIM

二等無人航空機操縦士の**実地審査（実技）練習用**ブラウザシミュレーターです。
国土交通省航空局の実地試験 実施基準・実施細則（回転翼航空機マルチローター・二等）に基づき、コース・区画・採点を再現しています。

> 本シミュレーターは練習補助を目的としたものであり、**実際の試験・修了審査の合否を保証するものではありません。**

---

## 動作環境

- 最新の **Google Chrome / Microsoft Edge**（WebGL・Gamepad API 対応）
- インストール不要・**完全オフライン動作**

## 使い方

1. `index.html` `dronepeak_flightsimV1.0.html` `three.min.js` `emblem.png` を**同じフォルダ**に置きます。
2. `index.html`（タイトル画面）を開き、配布された**アクティベーションキー**を入力して認証します。
3. 「シミュレーターを起動」を押すとシミュレーター本体が開きます。
4. ゲームパッド／プロポを接続すると自動認識します（未接続でもキーボードで操作可）。

※ ファイルは必ずセットで配置してください（HTMLが隣の `three.min.js` を読み込みます）。

> **配布者（管理者）向け**：キーの追加・管理は [`再配布防止_対策メモ.md`](再配布防止_対策メモ.md) を参照。
> キー作成ツール（`_admin_keygen.html`）は手元の管理用フォルダに別置きしてあり、**GitHubには公開（アップロード）しません**。

## 操作

- **入力モード**：モード1／モード2（メニューで切替。既定はモード2）
- **スティック割り当て**：標準プリセット、または「オート割り当て」で実機の軸を読み取って自動設定
- **キーボード**：スロットル `W/S`、エレベーター `↑/↓`、ラダー `A/D`、エルロン `←/→`、**Enter＝実地審査開始 / Space＝停止**
- **プロポ軸スイッチ**：軸4＝実地審査開始 / 軸5＝リセット

## 機能

- 種目：スクエア飛行 / 8の字飛行 / 異常事態における飛行
- 限定変更：基本（目視内・昼間）/ 目視外（FPV表示＋直上ナビ）/ 夜間（暗所＋機体灯火・ヘリパッド照明）
- 講師ナビ（音声＋テキスト案内）、推奨飛行ガイドライン、環境風（風速・風向き）
- 自動採点（100点持ち点・減点式・70点合格）、減点ホイッスル／不合格ブザー、制限時間カウントダウン
